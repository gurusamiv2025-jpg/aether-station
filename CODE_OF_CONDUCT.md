# Code of Conduct

This project follows the **Contributor Covenant v2.1** in spirit.

## Our pledge

We pledge to make participation in this project a harassment-free
experience for everyone, regardless of age, body size, visible or
invisible disability, ethnicity, sex characteristics, gender identity
and expression, level of experience, education, socio-economic status,
nationality, personal appearance, race, religion, or sexual identity
and orientation.

## Standards we expect

- Be welcoming, patient, and helpful in code review and discussion.
- Assume good faith; ask questions before assigning motive.
- Use specific feedback ("this function has an off-by-one" beats "this
  is bad").
- Disagree about the work, not the person.

## Behavior we won't accept

- Personal attacks, harassment, or discriminatory language.
- Sexualized comments or imagery.
- Publishing other people's private information without consent.
- Trolling, deliberate misrepresentation, or pile-ons.

## Reporting

Open a private GitHub Security Advisory or contact the maintainer
listed in `pyproject.toml`. Reports will be reviewed and acted on
proportionally.

## Scope

This Code of Conduct applies to all project spaces — repository, issues,
PRs, discussions, and any in-person or online events organized around
the project.
