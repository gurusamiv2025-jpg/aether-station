feel for the cast without spinning up the app. Replies below were

## Reasoning trace (collapsed in the UI)

Every reply ships with a five-step trace:

| Step | Detail |
|---|---|
| 1. Intent | Tags inferred from the question (e.g. `technical`, `incident`, `personal`). |
| 2. Query → Foundry IQ | What the knowledge layer was asked, with intent hints. |
| 3. Retrieved | Lore files returned, with relevance scores. |
| 4. Salient facts | Sentences from the passages with the highest token overlap to the question. |
| 5. Character synthesis | LLM composes the reply in the character's voice using only the salient facts. |
