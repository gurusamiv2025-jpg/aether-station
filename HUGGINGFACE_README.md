# How to use this Space

This is the Hugging Face Spaces presentation for **Aether Station** — a
multi-character chatbot grounded in a station lore bible via Foundry IQ.

Set secrets under **Settings → Variables and secrets** if you want
live Azure OpenAI / Foundry IQ. Otherwise the app runs in offline demo
mode using local retrieval + a deterministic per-character mock LLM.

See the main README in the repo for full details and the
[GitHub source](https://github.com/USERNAME/aether-station).

**When publishing to a Space, copy this file to `README.md` inside the
Space repo (Spaces require the YAML front-matter at the top of README).**
